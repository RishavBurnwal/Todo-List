# Todo List

A simple, lightweight Todo List web app built with HTML, CSS, and JavaScript.

Features
- Add, complete, and remove tasks
- Tasks are saved in localStorage so they persist across page reloads
- Responsive, minimal UI

Getting started
1. Clone the repo:
   git clone https://github.com/RishavBurnwal/Todo-List.git
2. Open index.html in your browser, or use a live server extension.

Usage
- Type a task into the input and press Enter or click Add.
- Click a task to toggle completion.
- Click the delete button/icon to remove a task.
